# LightBaker3000 Frontend

## Описание программы

Вспомогательный редактор сцены для утилиты LightBaker3000 за авторством тов. ncuxonaT

## Возможности

- Загрузка сцены из модели формата .obj
- Загрузка карт в формате BSP (Goldsource/Xash3d)
- Редактирование entity для карт в формате BSP
- Поддержка нескольких мониторов и произвольная конфигурация вьюпортов (на манер Blender)
